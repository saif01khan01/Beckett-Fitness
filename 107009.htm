Project Summary
Please give a brief summary of your study (maximum 100 words). 
The aim of this project will be to develop a mobile application that allows a user to efficiently track the calories in their meals. This will allow for them to be conscious in the way they eat and allows them to keep an eye on what they eat throughout the day. The project will be developed as a mobile application using Java as the main language for the development. The application will also be hosted somewhere using cloud technologies.
